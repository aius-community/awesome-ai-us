## ✋ How to Submit / 如何提交

We'd love to feature your project! Here's how:

我们很想展示你的项目！方法如下：

**Option 1: Open an Issue（推荐新手）**

1. Go to [Issues](https://github.com/aius-community/awesome-ai-us/issues) → Click **New Issue**
2. Title: Your project name / 你的项目名
3. Include the following / 请包含以下信息：
   - Project name and link / 项目名称和链接
   - One-line description in English & Chinese / 一句话描述（中英文）
   - Which AI tools you used / 用了哪些 AI 工具
   - Your GitHub username / 你的 GitHub 用户名

**Option 2: Submit a Pull Request（适合熟悉 GitHub 的人）**

1. Fork this repo
2. Add your project to the appropriate category in this README
3. Submit a PR

## 📜 Submission Guidelines / 提交规则

- ✅ Project must be built with the help of AI / 项目必须借助 AI 工具完成
- ✅ You don't need to be a programmer / 你不需要是程序员
- ✅ Any stage is welcome: prototype, MVP, finished product / 任何阶段都欢迎：原型、MVP、成品
- ✅ Brief description in both English & Chinese preferred / 建议提供中英文简介
- ❌ No spam, no purely AI-generated content without human input / 不接受垃圾内容
